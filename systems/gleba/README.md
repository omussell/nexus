# GLEBA 

Keeps track of statistics associated with data, like number of citations associated with a journal article

As data is processed we should create metrics about the data to make it easier to find patterns.

These statistics can also be used for billing purposes, for example, you may charge based on the amount of new data added per organization.

This statistics data "spoils" or becomes stale because it needs to be regularly updated and causes problems if the data is old.
